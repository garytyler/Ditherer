# Development

## Unity Editor Version
Unity 5.4.0f3
https://unity3d.com/unity/whats-new/unity-5.4.0

## Accompanying Asset Downloads
https://www.dropbox.com/sh/lh79g5uj3vkkfi4/AAC16O3apSCMVRzG9icIDG_ca?dl=0
Place all folders in /Assets folder:
/Assets/StreamingAssets
/Assets/Textures-Other
/Assets/Textures-Substance

## Hap Video Codec
https://github.com/Vidvox/hap
This is an open-source codec for GPU based video playback. It is used by the AVPRO Video plugin, so only needed to view 360 video experiences.

## Project Structure
### /Audio
All audio files go here
### /Animations
All Animation Object files for scenespace animations go here.
### /Scenes
All Unity Scenes for this project go here
### /Scripts
All custom scripts go here
### /Materials-Util
Utility materials, not for use in final product
### /Models
.FBX models, including a /Materials folder
### /SceneAssets
Mostly older models and materials but some still in use. Use /Texture,/Model and other folders for new assets.
### /StreamingAssets
Video files for playback by AVPro Video plugin, Product scene experience videos
### /Textures-Other
Texture files not originated from Substance tools
### /Textures-Substance
Texture files originated from Substance tools
